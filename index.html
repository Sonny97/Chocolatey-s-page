Hablemos un poco de chocolatey

Que es?

Chocolatey es un manejador de paquetes(programas) desde la consola de windows CMD o Power Shell.
La idea principal de la creacion de chocolatey fue dar el facil uso de la consola para instalacion de packetes como lo hace apt-get o yum en linux o centus7

Para que sirve

Con el podemos instalar paquetes que normalmente en windows puede ser un poco tediosa la instalacion mientras que con otros sistemas operativos bastaba con una sola linea de codigo, y esperar a que se instalara, Chocolatey esta aqui para mejorar vidas, para mejorar tu vida en aras de la libertad... Abraham Linconl

Como instalarlo

la instalacion consta de:
  • entrar a Chocolatey
  • Copiar el codigo @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
